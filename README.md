🚗 Bright Motors Data Analysis Project

📌 Problem Statement

Bright Motors aims to better understand its vehicle sales data in order to improve decision-making, increase profitability, and identify key market trends. However, the raw dataset contains inconsistencies such as missing values, unstructured date formats, and duplicate records (e.g., VIN duplication), making it difficult to extract reliable insights.

---

🎯 Aim of the Project

The goal of this project is to clean, transform, and analyze vehicle sales data to uncover meaningful insights that can support business strategy, optimize sales performance, and enhance customer understanding.

---

🧩 Objectives & Steps Taken

1. Data Understanding & Planning

* Reviewed dataset structure and key fields (VIN, selling price, sale date, etc.)
* Defined key metrics such as total revenue and profit margin

2. Data Cleaning & Transformation

* Removed duplicate records using VIN as a unique identifier
* Handled missing values and standardized categorical fields
* Converted text-based numerical values (e.g., "15,000") into numeric format
* Parsed and formatted date columns into proper date/timestamp format
* Created calculated columns:

  * Total Revenue = Selling Price × Units Sold
  * Profit Margin = (Selling Price – Cost Price) / Selling Price

 3. Data Analysis

* Analyzed revenue by car make and model
* Explored relationships between price, mileage, and year of manufacture
* Identified top-performing regions based on sales volume
* Examined trends in customer purchasing behavior over time

 4. Data Visualization & Reporting

* Built interactive dashboards to present key insights
* Used filters (region, fuel type, year) for dynamic analysis
* Presented findings with clear visuals and business recommendations

---

 📊 Summary of Results

* Identified top-performing car brands and models contributing the highest revenue
* Discovered trends in pricing and vehicle age affecting sales performance
* Highlighted regions with the highest sales activity
* Provided actionable insights to improve pricing strategies and inventory decisions

---

🛠️ Tools Used

* Miro – Project planning and data flow design
* Canva - Gantt chart timeline
* SQL Databricks – Data cleaning and transformat
* Microsoft Excel – Data visualization and dashboards
* PowerPoint – Presentation and storytelling
---

 📈 Conclusion

This project demonstrates how structured data cleaning, analysis, and visualization can transform raw data into actionable insights that support strategic decision-making in the automotive sales industry.
